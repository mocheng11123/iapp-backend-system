## 三、界面设计详解

### 3.1 登录页（/login）

**界面元素**：
- 居中卡片布局
- 渐变背景
- 邮箱输入框
- 密码输入框（显隐切换）
- 登录按钮（带 loading 状态）
- API 文档链接
- 注册入口

**交互流程**：
1. 输入邮箱密码
2. 回车或点击登录
3. 调用 `/dev/login` 获取 token
4. 存储 token 到 localStorage
5. 跳转到仪表盘

## 五、部署方案

### 5.1 开发模式

```bash
# 后端
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 前端
npm run dev
```

前端访问 http://localhost:3000，代理转发 API 请求。

### 5.2 生产模式

**方案 A：前后端分离部署**
```
admin.yourdomain.com  ->  Nginx 静态文件（Vue 构建产物）
api.yourdomain.com    ->  Nginx 反向代理到 FastAPI
```

**方案 B：同域部署**
```
yourdomain.com/       ->  Vue 静态文件
yourdomain.com/dev/*  ->  FastAPI
yourdomain.com/api/*  ->  FastAPI
```

### 5.3 Nginx 配置（方案 B）

```nginx
server {
    listen 80;
    server_name yourdomain.com;

    # 前端静态文件
    location / {
        root /var/www/admin;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # API 代理
    location /dev {
        proxy_pass http://backend:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location /api {
        proxy_pass http://backend:8000;
        proxy_set_header Host $host;
    }

    # API 文档
    location /docs {
        proxy_pass http://backend:8000/docs;
    }
}
```
